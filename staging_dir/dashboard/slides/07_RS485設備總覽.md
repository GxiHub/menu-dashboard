# RS485 設備總覽 — 機械手臂通訊協議

**建立日期：** 2026-03-06　｜　**狀態：** 🟢 測試中

---

## 匯流排架構

| 項目 | 內容 |
|------|------|
| 協議 | RS485 Modbus-RTU |
| 串口 | `/dev/ttyACM0` |
| 通訊格式 | 8N1（8 data / no parity / 1 stop） |
| 主控 | Raspberry Pi 5（pi53） |
| 裝置數 | 5 台（A/B/C/D/E 軸） |
| Web UI | `http://192.168.1.111:5001` |

---

## 軸配置總表

| 軸 | Slave ID | 品牌 | 波特率 | 功能 | 關節類型 |
|----|----------|------|--------|------|----------|
| **A** | 2 | 英鵬飛 | 115200 | 底座旋轉 Base | Yaw（水平） |
| **B** | 1 | 另品牌 | 9600 | 肩關節 Shoulder | Pitch（垂直） |
| **C** | 3 | 英鵬飛 | 115200 | 肘關節 Elbow | Pitch（垂直） |
| **D** | 4 | 英鵬飛 | 115200 | 腕部旋轉 Wrist | Yaw |
| **E** | 5 | 英鵬飛 | 115200 | 末端姿態 Wrist | Pitch/Roll |

> **注意**：B 軸波特率 9600 與其他軸 115200 不同，`arm_driver.py` 預設 9600 共用串口。

## A/C/D/E 軸 — 英鵬飛步進控制器

### 核心寄存器

| 寄存器 | 名稱 | 類型 | R/W | 說明 |
|--------|------|------|-----|------|
| `0x009A` | 運行速度 | WORD | R/W | 0~10000，建議 300~1200 |
| `0x00C8` | 運行動作 | WORD | W | 控制指令（見下表） |
| `0x00CE~0x00CF` | 相對位置運行 | DWORD | W | 正負表示方向，脈衝數 |
| `0x00CC~0x00CD` | 按時間運行 | DWORD | W | 單位 ms，正負方向 |
| `0x00CA` | 點動控制 | WORD | W | 方向 + 點動速度 |
| `0x006E~0x006F` | 軟負限位 | DWORD | R/W | 負方向最大脈衝 |
| `0x0070~0x0071` | 軟正限位 | DWORD | R/W | 正方向最大脈衝 |
| `0x009B` | 硬體限位配置 | WORD | R/W | NPN/PNP 設定 |
| `0x0082` | 告警碼 | WORD | R | 0=無告警 |
| `0x0006` | IO 輸入狀態 | WORD | R | bit0 = Hall 感測器 |

### 運行動作指令（0x00C8）

| 值 | 功能 |
|----|------|
| `0` | 減速停止 |
| `1` | 正向運行 |
| `257` (0x0101) | 反向運行 |
| `256` (0x0100) | 急停 |

### 表格控制（簡易程序）

| 寄存器 | 名稱 | 說明 |
|--------|------|------|
| `0x00AA` | 表大小 | 表格長度 |
| `0x00AB` | 表指針 | 當前執行位置 |
| `0x00AC` | 表起始地址 | 表資料起始 |
| `0x00DD` | 執行表格 | 啟動表格運行 |

### Modbus 幀格式

```
FC03 讀寄存器：[Slave][03][RegH][RegL][CountH][CountL][CRC_L][CRC_H]
FC06 寫寄存器：[Slave][06][RegH][RegL][DataH][DataL][CRC_L][CRC_H]
FC10 寫多個：  [Slave][10][RegH][RegL][CountH][CountL][ByteN][Data...][CRC_L][CRC_H]
```

### Python 控制範例

```python
from arm_driver import RobotArm

arm = RobotArm("/dev/ttyACM0", 9600)
arm.a.set_speed(300)       # A 軸設速度
arm.a.jog_forward()        # A 軸正轉
arm.a.stop()               # A 軸停止
arm.a.move_relative(1600)  # A 軸正向移動 1600 脈衝
arm.a.read_alarm()         # 讀告警碼
arm.e.read_hall()          # E 軸讀 Hall 感測器
arm.estop_all()            # 全軸急停
arm.close()
```

## B 軸 — 另品牌步進控制器

### 核心寄存器

| 地址 | 名稱 | 說明 | 範圍 |
|------|------|------|------|
| `0x01` | Direction | 1=正轉, 0=反轉 | 0/1 |
| `0x02` | Run/Pause | 1=運行, 0=暫停 | 0/1 |
| `0x03` | Stop | 停止並清行程 | 寫1 |
| `0x04` | Speed | 轉速 | 1~800 RPM |
| `0x05` | Pulse Count | 運動脈衝數 | 0~65535 |
| `0x06` | Turn Count | 圈數（0=連續轉） | 0~65535 |
| `0x07` | Angle | 目標角度 | 0~65535 |
| `0x08` | Device Address | Modbus 站號 | 1~247 |
| `0x09` | Offline Enable | 釋放電機 | 0/1 |
| `0x0A` | One-key Home | 一鍵回原點 | — |
| `0x0B` | Power-on Homing | 上電歸零 | 0/1 |
| `0x0C` | Homing Direction | 歸零方向 | 1正/0反 |
| `0x0D` | Limit Enable | 啟用限位開關 | 0/1 |
| `0x0E` | Acceleration | 加速度 | 0~10 |
| `0x0F` | Baudrate | 波特率設定 | 見下表 |
| `0x15` | Set Home | 設當前位置為原點 | 寫1 |
| `0x16` | Current Angle | 已運行角度（有符號） | R |
| `0x17` | Limit Status | 限位開關狀態 | R |
| `0x1A` | Home Speed | 回原點速度 | 1~400 RPM |

### 波特率對照

| 值 | 波特率 |
|----|--------|
| 0 | 1200 |
| 1 | 2400 |
| 2 | 4800 |
| **3** | **9600（預設）** |
| 4 | 19200 |

### 限位開關狀態（0x17）

| 值 | 說明 |
|----|------|
| 0 | 無觸發 |
| 1 | 正轉方向觸發 |
| 2 | 反轉方向觸發 |
| 3 | 正反兩側均觸發 |

### 控制指令範例（Slave=1）

```
正轉：       01 06 00 01 00 01 19 CA
反轉：       01 06 00 01 00 00 B8 0A
開始運行：   01 06 00 02 00 01 E9 CA
暫停：       01 06 00 02 00 00 28 0A
停止清行程： 01 06 00 03 00 01 B8 0A
設速 120RPM：01 06 00 04 00 78 C8 29
設 1600 脈衝：01 06 00 05 06 40 9B 9B
設 5 圈：    01 06 00 06 00 05 A9 C8
設 360 度：  01 06 00 07 01 68 38 75
釋放電機：   01 06 00 09 00 01 98 08
鎖定電機：   01 06 00 09 00 00 59 C8
```

### Python 控制範例

```python
arm.b.set_speed(60)            # 設速度 60 RPM
arm.b.jog_forward()            # 正轉（連續）
arm.b.jog_reverse()            # 反轉（連續）
arm.b.pause()                  # 暫停（保留行程）
arm.b.stop()                   # 停止（清行程）
arm.b.move_revolutions(5)      # 正轉 5 圈
arm.b.move_angle(180)          # 移動 180 度
arm.b.set_home()               # 設當前為原點
arm.b.go_home()                # 一鍵回原點
arm.b.read_angle()             # 讀已運行角度
arm.b.read_limit_status()      # 讀限位狀態
arm.b.enable_limit_switch()    # 啟用限位
```

## 感測器

### Hall 感測器（已確認 2026-03-06）

| 項目 | 內容 |
|------|------|
| 安裝位置 | E 軸 |
| 讀取方式 | Slave 5, 寄存器 `0x0006`, bit 0 |
| 觸發 | bit0=1（磁鐵靠近, 綠燈亮） |
| 釋放 | bit0=0（磁鐵移開, 綠燈滅） |
| 訊號品質 | 穩定，無彈跳 |
| 用途 | 歸零偵測 / 限位 |

## 運動控制流程

```
1. set_speed()           設定目標速度
2. set_soft_limits()     設定軟限位範圍（防撞）
3. jog / move_relative   手動微調 或 脈衝定位
4. read_hall()           監控 Hall 感測器（歸零）
5. estop_all()           異常時全軸急停
```
