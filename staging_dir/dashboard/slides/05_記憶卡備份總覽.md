# Pi5 記憶卡備份總覽

**建立日期：** 2026-03-05　｜　**狀態：** 🟢 運作中（每 6 小時自動備份）

---

## 一句話摘要

> SD 卡壞掉時，靠 **GitHub 7 個 Repo** + **~/secrets 手動還原** + **~/RECOVERY.md 步驟**，可在 30 分鐘內重建整台 Pi5。

---

## 備份架構總覽

```
Pi5 記憶卡（57GB，已用 16GB）
│
├─ 程式碼（7 個 Git Repo）─── github_backup.sh 每 6 小時 ──→ GitHub (GxiHub)
│   ├── dashboard          → GxiHub/menu-dashboard
│   ├── piupload           → GxiHub/piupload
│   ├── raspi-system       → GxiHub/raspi-system
│   ├── fb_scraper         → GxiHub/fb-scraper
│   ├── accounting         → GxiHub/luwei-accounting
│   ├── projects/robot_arm → GxiHub/robot-arm
│   └── dotfiles           → GxiHub/raspi-dotfiles ← NEW
│       ├── 21 個 home 腳本/文件
│       ├── bashrc + crontab.txt
│       └── db_dumps/          ← SQLite 自動匯出
│           ├── menu.sql        （菜單 95 筆）
│           ├── accounting.sql  （記帳資料）
│           └── fb_posts.sql    （爬蟲資料）
│
├─ 機密檔（~/secrets/）─── ⚠️ 不進 GitHub，需手動備份
│   ├── telegram_env             → symlink 到 ~/dashboard/.telegram_env
│   ├── fb_accounts.json         → symlink 到 ~/fb_scraper/accounts.json
│   ├── cloudflared_config.yml   → /etc/cloudflared/config.yml 副本
│   └── cloudflared_credential.json → Tunnel 憑證副本（最重要！）
│
└─ 可重建（不需備份）
    ├── Docker images     → docker compose build 重建
    ├── releases/         → 重新部署即產生
    ├── logs/             → 歷史日誌，可捨棄
    └── node_modules/     → npm install 重建
```

## 自動備份流程（每 6 小時）

```
github_backup.sh（cron: 0 */6 * * *）
│
├── Phase A：SQLite 資料庫匯出
│   ├── menu.db      → dotfiles/db_dumps/menu.sql
│   ├── accounting.db → dotfiles/db_dumps/accounting.sql
│   └── posts.db     → dotfiles/db_dumps/fb_posts.sql
│
├── Phase B：Home 檔案同步
│   ├── 13 個 .sh/.py 腳本 → dotfiles/
│   ├── 7 個 .md 文件       → dotfiles/
│   ├── ~/.bashrc           → dotfiles/bashrc
│   └── crontab -l          → dotfiles/crontab.txt
│
└── Phase C：Git commit + push（7 個 repo）
    └── 有變更才 commit，無變更跳過
```

## 機密檔管理

| 檔案 | ~/secrets/ 內名稱 | 原始位置 | 連結方式 |
|------|-------------------|---------|---------|
| Telegram Token | `telegram_env` | `~/dashboard/.telegram_env` | symlink |
| FB 帳號 | `fb_accounts.json` | `~/fb_scraper/accounts.json` | symlink |
| Tunnel 設定 | `cloudflared_config.yml` | `/etc/cloudflared/config.yml` | 副本 |
| Tunnel 憑證 | `cloudflared_credential.json` | `/etc/cloudflared/*.json` | 副本 |

⚠️ **~/secrets/ 不進 Git**（.gitignore 設為 `*`），需另外備份：
- 方法 A：截圖/抄錄到密碼管理器
- 方法 B：`tar czf secrets_backup.tar.gz ~/secrets/` 加密保存
