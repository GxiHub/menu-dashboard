## 自動備份流程（每 6 小時）

```
github_backup.sh（cron: 0 */6 * * *）
│
├── Phase A：SQLite 資料庫匯出
│   ├── menu.db      → dotfiles/db_dumps/menu.sql
│   ├── accounting.db → dotfiles/db_dumps/accounting.sql
│   └── posts.db     → dotfiles/db_dumps/fb_posts.sql
│
├── Phase B：Home 檔案同步
│   ├── 13 個 .sh/.py 腳本 → dotfiles/
│   ├── 7 個 .md 文件       → dotfiles/
│   ├── ~/.bashrc           → dotfiles/bashrc
│   └── crontab -l          → dotfiles/crontab.txt
│
└── Phase C：Git commit + push（7 個 repo）
    └── 有變更才 commit，無變更跳過
```

```bash
# 製作備份包（792 bytes，存到安全的地方）
cd ~ && tar czf secrets_backup.tar.gz secrets/
```

## 一鍵還原（SD 卡壞掉時）

### 事前：你只需要準備兩樣東西

1. **一張新 SD 卡**（燒好 Raspberry Pi OS，user 設為 `pi53`）
2. **`secrets_backup.tar.gz`**（792 bytes，從密碼管理器或備份處取得）

### 還原：兩條指令搞定

```bash
# 1. 把 secrets_backup.tar.gz 傳到新 Pi5
scp secrets_backup.tar.gz pi53@<new-pi-ip>:~/

# 2. SSH 進新 Pi5，下載並執行還原腳本
curl -fsSL https://raw.githubusercontent.com/GxiHub/raspi-dotfiles/main/restore_pi5.sh -o restore_pi5.sh
bash restore_pi5.sh
```

### 腳本自動完成的 12 個步驟

```
 [ 1/12] 安裝系統套件（Docker, gh CLI, tmux...）
 [ 2/12] GitHub 認證（互動式登入）
 [ 3/12] Clone 7 個 Repo
 [ 4/12] 還原 Home 腳本和文件（21 個）
 [ 5/12] 還原機密檔（~/secrets/ 解壓 + symlink）
 [ 6/12] 還原 Cloudflare Tunnel 設定
 [ 7/12] 建立 Docker Network
 [ 8/12] 啟動 Docker 服務（build + up）
 [ 9/12] 還原 SQLite 資料庫（3 個 .sql → .db）
 [10/12] 還原 Crontab（7 條排程）
 [11/12] 安裝 Tailscale + Claude Code
 [12/12] 驗證所有服務狀態
```

腳本位置：`~/dotfiles/restore_pi5.sh`（GitHub: `GxiHub/raspi-dotfiles`）
