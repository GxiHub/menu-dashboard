# 電動升降滷台系統

**類型：** 智慧餐飲設備　｜　**版本：** v1.2　｜　**建立：** 2026-03-04　｜　**更新：** 2026-03-09

---

## 設計目標

> 自動化滷台：透過 Raspberry Pi 控制電動推桿升降滷籃，搭配 5000W 電磁加熱與 PT100 溫度監控，使用 RS485 控制整個系統。

解決的問題：人工操作重複、食材浸泡時間不穩定、人力成本高、操作效率低。

## 四、系統組成

| 系統 | 核心設備 | 規格 |
|------|----------|------|
| **加熱系統** | 電磁加熱機芯 + 感應線圈 | 5000W / 30×30 線圈 |
| **升降機構** | 電動推桿 + 升降平台 | 6支 24V 同步 |
| **控制系統** | Raspberry Pi 5 | UI / 流程控制 / 網路 |
| **通訊系統** | RS485 工業通訊 | 繼電器 + 溫度 + 中繼器 |
| **感測系統** | PT100 溫度感測器 | 即時監測 / 過熱保護 |

## 五、自動化料理流程

```
加熱湯鍋  →  放入食材  →  推桿下降
                              │
完成料理  ←  推桿升起  ←  浸泡計時
```

**控制邏輯：** 時間 + 溫度雙重條件，確保每次料理品質一致。

## 六、安全設計

- 接地保護（220V 高壓側）
- 推桿限位開關（防止超程）
- PT100 即時溫度監測（過熱自動斷電）
- 繼電器控制隔離（低壓控制高壓）

## 八、1 分鐘介紹

> 這是一台自動化滷台，透過 **Raspberry Pi** 控制 **電動推桿** 升降滷籃，搭配 **5000W 電磁加熱** 與 **PT100 溫度監控**，使用 **RS485** 控制模組來管理整個系統。
>
> 可以自動控制食材浸泡時間、穩定料理品質，同時降低人力操作，未來也可以接入 AI 或雲端系統。

## 九、Web UI（pi52 控制介面）

| 項目 | 內容 |
|------|------|
| 主機 | Raspberry Pi（pi52） |
| 服務 | Flask app.py |
| 路徑 | `/var/www/html/` |
| 版控 | Git（`/var/www/html/.git`） |

### 功能區塊

| 功能 | 說明 |
|------|------|
| 電磁爐控制 | 手動功率調整 / 自動溫控 ON/OFF |
| PT100 溫度 | 即時顯示、升溫速率 |
| 加熱計時器 | 設定水量 → 一鍵開始，自動連動自動溫控 + 風扇 |
| 風扇控制 | 三路風扇獨立開關（風扇 3 = 繼電器 CH4） |
| 電動推桿 | 多組推桿升降控制 |
| 設定面板 | 加熱功率 / 緩衝溫度 / 保溫溫度 |
| 全畫面切換 | 左下角 ⛶ FAB，適合無鍵盤觸控操作 |

### 繼電器通道對照（pi52 FAN_MAP）

| 風扇 | RS485 Coil | 備註 |
|------|-----------|------|
| 風扇 1 | CH6 (0x0C) | 手動 |
| 風扇 2 | CH7 (0x0C) | 手動 |
| 風扇 3 | CH4 (0x0C) | 自動溫控連動 |

### 一鍵加熱流程

```
設定水量（最大 30L）
        ↓
按「開始」
        ↓
自動溫控 ON ──→ 系統自動調整電磁爐功率
風扇 3 ON   ──→ 散熱 / 對流
計時器啟動  ──→ 顯示已加熱 / 預估剩餘 / 升溫速率
        ↓
達到 100°C
        ↓
自動溫控 OFF
風扇 3 OFF
計時器停止，顯示完成
```
