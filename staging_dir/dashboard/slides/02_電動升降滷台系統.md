## 設計目標

> 自動化滷台：透過 Raspberry Pi 控制電動推桿升降滷籃，搭配 5000W 電磁加熱與 PT100 溫度監控，使用 RS485 控制整個系統。

解決的問題：人工操作重複、食材浸泡時間不穩定、人力成本高、操作效率低。

## 五、自動化料理流程

```
加熱湯鍋  →  放入食材  →  推桿下降
                              │
完成料理  ←  推桿升起  ←  浸泡計時
```

**控制邏輯：** 時間 + 溫度雙重條件，確保每次料理品質一致。

## 七、Web UI（pi52 控制介面）

| 項目 | 內容 |
|------|------|
| 主機 | Raspberry Pi 5（pi52） |
| 服務 | Flask app.py |
| 路徑 | `/var/www/html/` |
| 本地 IP | 192.168.0.21:5000 |
| Tailscale | 100.98.225.85:5000 |

### 功能區塊（v2026-03-11 後）

| 功能 | 說明 |
|------|------|
| 系統狀態 | 序列埠 / 溫度感測器 / 繼電器 / 電磁爐 連線狀態，一鍵診斷 |
| 電磁爐控制 | 手動功率調整（關閉/20%/50%/65%/70%）/ slider |
| PT100 溫度 | 即時顯示、IGBT 溫度、回讀功率、錯誤碼 |
| 電動推桿 | 多組推桿升降控制（主要操控區塊） |
| 自動溫控 | 四段式自動控溫 ON/OFF，六個參數可調 |
| 溫度曲線 | 獨立頁面（/temp_curve），記錄 session 數據 |

### 四段式自動溫控邏輯（v2026-03-11）

| 階段 | 條件 | 功率 |
|------|------|------|
| 加熱期（低段） | T < 89°C | 65% |
| 加熱期（衝刺） | 89°C ≤ T < 100°C | 70% |
| 保溫期 | T ≥ 100°C | 40% |
| 保溫補熱 | 保溫中 T < 93°C | 70% |

### RS485 繼電器模組配置

| 模組 | 地址 | 用途 |
|------|------|------|
| 繼電器模組 A | 0x0B | 電動推桿（CH0-CH7，配對控制） |
| 繼電器模組 B | 0x0C | 風扇（CH4-CH7，風扇1-3） |

## 八、1 分鐘介紹

> 這是一台自動化滷台，透過 **Raspberry Pi 5（pi52）** 控制 **電動推桿** 升降滷籃，搭配 **5000W 電磁加熱** 與 **PT100 溫度監控**，使用 **RS485** 控制模組來管理整個系統。
>
> 四段式自動溫控確保水溫穩定到達並維持 100°C，可以自動控制食材浸泡時間、穩定料理品質，同時降低人力操作，未來也可以接入 AI 或雲端系統。
