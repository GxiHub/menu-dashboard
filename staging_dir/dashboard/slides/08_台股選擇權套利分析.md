## 這個系統在做什麼？

從**台灣期貨交易所（TAIFEX）**自動抓取臺指選擇權（TXO）每日行情，用 4 種策略分析是否存在**套利空間**或**低風險交易機會**，盤後自動推送 Telegram 報告。

---

## 選擇權基礎觀念

| 名詞 | 說明 |
|------|------|
| **買權 (Call)** | 付權利金，取得「未來用某價格買進」的權利 |
| **賣權 (Put)** | 付權利金，取得「未來用某價格賣出」的權利 |
| **履約價 (Strike)** | 合約約定的買賣價格 |
| **權利金 (Premium)** | 買選擇權要付的價格 |
| **價外 (OTM)** | Call 履約價 > 現貨、Put 履約價 < 現貨（沒有內含價值） |
| **隱含波動率 (IV)** | 市場預期未來波動程度，IV 越高權利金越貴 |

> 簡單說：選擇權就是「買保險」。Call 是怕漲的保險，Put 是怕跌的保險。
> 我們的策略核心 = **當保險公司**，賣保險收權利金。

## 策略一：Put-Call Parity 套利

### 原理

金融理論告訴我們，同一履約價、同一到期日的 Call 和 Put 有固定的價格關係：

```
Call價 - Put價 = 現貨價 - 履約價 × e^(-rT)
```

如果市場價格偏離這個公式 → 代表有套利機會。

### 怎麼賺？

| 情況 | 操作 | 意思 |
|------|------|------|
| **Call 偏貴** | 賣 Call + 買 Put + 買期貨 | 鎖定 Call 的溢價 |
| **Put 偏貴** | 買 Call + 賣 Put + 賣期貨 | 鎖定 Put 的溢價 |

### 分析方法

1. 從期交所抓所有 Call / Put 配對
2. 用實際 **bid/ask 報價**（非結算價）計算可執行價差
3. 扣除 3 腳交易成本（每腳 15 點 × 3 = 45 點）
4. 只保留**有成交量**的合約（能實際交易的）

### 今日結果（2026/03/06）

找到 **44 個有流動性的機會**，前 3 名：

| 到期 | 履約價 | 淨利 | C量 | P量 | 操作 |
|------|--------|------|-----|-----|------|
| 202605 | 34000 | 111 pt | 10 | 10 | 賣C+買P+買現貨 |
| 202604 | 33000 | 62 pt | 3 | 20 | 賣C+買P+買現貨 |
| 202603W2 | 33850 | 52 pt | 153 | 40 | 賣C+買P+買現貨 |

> PCP 套利需要**同時**建立 3 個部位，速度是關鍵，適合程式化交易。

## 策略二：Box Spread 套利

### 原理

同時做兩個方向相反的價差：

```
Box = Bull Call Spread(K1,K2) + Bear Put Spread(K1,K2)
    = 買C(K1) + 賣C(K2) + 買P(K2) + 賣P(K1)
```

不管到期時指數在哪，Box 的價值恆等於 **K2 - K1**。

如果建倉成本 < K2 - K1 → 穩賺不賠。

### 今日結果

**無套利機會** — 市場定價效率高，散戶想靠 Box Spread 套利極難。

## 策略三：Iron Condor（賺時間衰減）

### 原理

同時賣出價外 Call 和價外 Put，賭「指數不會大漲也不會大跌」：

```
賣 Call(高履約價) → 收權利金，賭不漲過去
賣 Put(低履約價)  → 收權利金，賭不跌過去
買 Call(更高)     → 保護翼，限制虧損上限
買 Put(更低)      → 保護翼，限制虧損上限
```

### 圖解

```
     虧損 ◄──────────────────────────────► 虧損
           買P    賣P    現貨    賣C    買C
           K1     K2     S      K3     K4
            ▼      ▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲▼
                   ^^^^^^^^^^^^^^^^^^^^^^^^
                    這個區間內都賺（安全區）
```

### 分析方法

1. 篩選距離現貨 **3%~8%** 的價外合約
2. 只看有成交量的（確保能交易）
3. 用 **bid 價**計算實際可收到的權利金
4. 按到期月份分組推薦

### 今日最佳推薦（週選 W2，5 天到期）

| 賣C | 賣P | 總收入 | 安全區間 | C量 | P量 |
|-----|-----|--------|----------|-----|-----|
| 34650 (+3.1%) | 32550 (-3.1%) | 241 pt | 32550-34650 | 372 | 192 |
| 34650 (+3.1%) | 32500 (-3.3%) | 234 pt | 32500-34650 | 372 | 1139 |
| 34700 (+3.3%) | 32550 (-3.1%) | 231 pt | 32550-34700 | 1007 | 192 |

> 白話：5 天內加權指數在 32550~34650 之間（±3%），就賺 241 點 = **12,050 元**。

## 策略四：隱含波動率 (IV) 分析

### 原理

用 **Black-Scholes 公式**從市場價格反算出「市場認為未來會波動多少」。

- IV 偏高 → 權利金被高估 → **適合賣出**（均值回歸）
- IV 偏低 → 權利金被低估 → 適合買進

### 分析方法

1. 對每個合約用 Brent 數值法反算 IV
2. 按到期分組，計算 **Z 分數**（偏離中位數幾個標準差）
3. Z > 0.5 的合約標記為「IV 偏高，適合賣出」

### 今日 IV 概覽

| 到期 | 天數 | Call IV | Put IV | 觀察 |
|------|------|---------|--------|------|
| 202603W2 | 5 | 30.0% | 39.3% | Put IV 明顯偏高（恐慌溢價） |
| 202603 | 12 | 31.3% | 56.5% | 月選 Put 更貴 |
| 202604 | 40 | 27.9% | 39.1% | 遠月相對穩定 |
| 202609 | 194 | 23.5% | 34.3% | 長天期 IV 較低 |

> **Put IV 始終 > Call IV** = 市場對下跌的恐懼大於上漲的期待（IV Skew），正常現象。

## 策略調整建議（基於回測）

### IC 的問題與修正

| 問題 | 修正方向 |
|------|----------|
| ±3% 太窄，被突破率高 | 改用 **±5%**，犧牲權利金換勝率 |
| 趨勢行情連續虧 | 加入**趨勢過濾**：均線、VIX 判斷 |
| 沒有停損機制 | 加入**動態停損**：虧 > 2× 收入時平倉 |
| 每天都做太頻繁 | 改為**每週三建倉一次**（週選最佳進場點） |

### BPS 為何表現好？

1. 單邊策略（只做看不跌），避開上漲方向的虧損
2. -3% 距離在正常行情足夠安全
3. 有翼保護，最大虧損可控

### 實戰建議

> **先做 Bull Put Spread，不做 Iron Condor。**
>
> BPS 回測 100% 勝率、操作單純（只要 2 腳）、風險有限。
> 等累積經驗和資料後，再評估是否加入 Call 端做完整 IC。

## 市場情報系統（進場過濾器）

回測暴露了最大問題：**沒有趨勢判斷就盲目賣選擇權 = 被單邊行情碾壓。**

所以加入 6 個維度的市場訊號，組成綜合評分，決定「今天該不該做、做哪邊」。

### 訊號來源

| 訊號 | 資料源 | 怎麼看 |
|------|--------|--------|
| **期貨正逆價差** | 期交所台指期 | 正價差=偏多，逆價差=恐慌偏空 |
| **盤後盤收盤** | 期交所台指期 | 反映美股盤後，預示隔日方向 |
| **外資期貨淨部位** | 三大法人 | 外資是最大玩家，跟著走勝率高 |
| **選擇權 P/C Ratio** | 三大法人 | >1.5=極度避險，<0.7=積極做多 |
| **台積電趨勢** | 證交所 | 佔加權 ~30%，它跌指數必跌 |
| **加權波動率** | 自行計算 | 高波動不適合賣方策略 |

### 評分機制

每個指標評 -2 ~ +2 分，加總後判斷方向：

| 總分 | 方向 | 建議策略 |
|------|------|----------|
| +4 以上 | 強烈偏多 | Bull Put Spread（賣 Put 收錢） |
| +2 ~ +3 | 偏多 | BPS 或 IC 偏多配置 |
| -1 ~ +1 | 中性 | Iron Condor（雙邊賣） |
| -2 ~ -3 | 偏空 | Bear Call Spread 或觀望 |
| -4 以下 | 強烈偏空 | 不做賣方，觀望或買 Put 避險 |

### 今日實際結果（2026/03/06）

```

外資期貨  -2  大量偏空 -38,930 口
P/C      -1  偏空
台積電    0  盤整（-4.3%/5日，但未破均線）
技術面    0  盤整（10日區間 32829~35095）
波動率   -1  46.8% 年化（太高）
────────────────────
總分: -2/12  方向: 偏空
>>> 不適合賣 Put，觀望或做 Bear Call Spread
```

### 為什麼這很重要？

回顧回測中 IC 最大虧損那筆（-37,835 元，02/06 建倉）：

- 當天外資期貨大量偏空
- 但 IC 不管方向照賣 → 隔週暴漲 +6.3% 突破賣 Call

**如果當天有市場情報系統，綜合分數會是負的 → 建議觀望 → 這筆虧損就避開了。**

## 跨市場時間差分析（Lead-Lag）

### 核心觀念

不同市場對同一事件的反應速度不同，**先反應的 = 領先指標，還沒反應的 = 交易機會**。

### 四條傳導鏈

```
美股收盤        台期夜盤         台股開盤        台股收盤
(05:00)         (05:00)         (08:45)         (13:45)
  │               │               │               │
  └──美股漲跌──►──┘               │               │
                  └──夜盤反應──►──┘               │
                                   │               │
台積ADR收盤 ─────►── 台積台股 ──►── 加權指數 ──┘
(05:00)             (08:45)        (~30%權重)

VIX 變動 ──────────►── 台灣 IV（延遲 1~3 天）
期貨急漲急跌 ──►── 選擇權跟上（延遲 5~30 分鐘）
```

### 監控什麼？

| 訊號 | 看什麼 | 機會 |
|------|--------|------|
| **台積電 ADR vs 台股** | 漲跌幅差異 = 台股還沒反應的部分 | ADR 跌 3% 但台股才跌 1% → 明天再跌 2% |
| **美股 → 台股** | S&P 大漲/大跌 > 1.5% | 台指期/選擇權跟隨做 |
| **VIX → 台灣 IV** | VIX 飆但台灣 IV 還沒跟 | 不要賣選擇權（IV 要補漲） |
| **期貨 → 選擇權** | 期貨急漲但選擇權還沒跟 | 買價外 Call/Put（還便宜） |

### 今日實測（2026/03/06）

| 項目 | 結果 |
|------|------|
| 台積 ADR vs 台股 | ADR -1.0% vs 台股 -0.5%，未反應差 -0.48%，同步 |
| S&P 500 | -0.6%，影響有限 |
| VIX | 23.75（偏高但未恐慌），台灣 IV 33.5%，差值正常 |
| 期貨 vs 選擇權 | 期貨 +0.1%，無時間差 |
| **結論** | 各市場同步，無明顯套利機會 |

> 今天沒機會是正常的 — 時間差機會通常出現在**重大事件日**（FOMC、財報、地緣衝突），平常日各市場會快速同步。
