## 二、硬體架構

```
Raspberry Pi（pi52）
        │ USB
        ▼
  /dev/ttyRS485（USB→RS485）
  9600 baud, Modbus RTU
        │
        ├── Slave 0x01  電磁爐（5000W）
        ├── Slave 0x08  PT100 溫度感測器（CH1）
        ├── Slave 0x0B  繼電器模組一
        │               coil 0/1  → 推桿 1 伸/縮
        │               coil 2/3  → 推桿 2 伸/縮
        │               coil 4/5  → 推桿 3 伸/縮
        │               coil 6    → 風扇 1
        │               coil 7    → 風扇 2
        └── Slave 0x0C  繼電器模組二（待接）
                        coil 4    → 風扇 3 ★自動溫控連動
                        coil 6/7  → 推桿 4/5
                        coil 0/1  → 推桿 6
```

### 繼電器通道對照

| 功能 | 模組 | Coil | 備註 |
|------|------|------|------|
| 推桿 1 伸出 | 0x0B | 0 | FC05 0xFF00 |
| 推桿 1 縮回 | 0x0B | 1 | FC05 0xFF00 |
| 推桿 2 伸出 | 0x0B | 2 | |
| 推桿 2 縮回 | 0x0B | 3 | |
| 推桿 3 伸出 | 0x0B | 4 | |
| 推桿 3 縮回 | 0x0B | 5 | |
| 風扇 1 | 0x0B | 6 | 手動 |
| 風扇 2 | 0x0B | 7 | 手動 |
| 風扇 3 | 0x0C | 4 | **自動溫控連動** |

| 項目 | 內容 |
|------|------|

### 主要 API 端點

| 端點 | 方法 | 說明 |
|------|------|------|
| `/api/status` | GET | 溫度、電磁爐功率、自動溫控狀態、風扇狀態 |
| `/api/induction` | POST | 設定電磁爐功率（`{power: 0~100}`） |
| `/api/auto_temp` | POST | 開關自動溫控、設定加熱/緩衝/保溫參數 |
| `/api/fan` | POST | 控制風扇（`{fan: 1~3, on: bool}`） |
| `/api/temp_log` | GET | 取得溫度曲線記錄（最多 1800 筆，約 1 小時） |
| `/api/temp_log/clear` | POST | 清除溫度曲線 |
| `/api/versions` | GET | 版本清單 |
| `/api/versions/save` | POST | 儲存當前版本 |
| `/api/versions/restore` | POST | 還原指定版本 |

## 四、自動溫控邏輯

```
溫度 < 緩衝起點（預設 90°C）
        → 加熱功率（預設 78%）

緩衝起點 ≤ 溫度 < 保溫起點（預設 100°C）
        → 緩衝功率（預設 40%）

溫度 ≥ 保溫起點
        → 保溫功率（預設 15%）

最高功率上限：AUTO_MAX_PWR = 78%（硬限制）
```

---

## 五、一鍵加熱流程

```
設定水量（0.5L ~ 30L）
        ↓
按「▶ 開始」
        ↓
POST /api/temp_log/clear    ← 清除舊曲線
POST /api/auto_temp {enabled:true}  ← 自動溫控 ON
POST /api/fan {fan:3, on:true}      ← 風扇 3 ON
計時器啟動 + 溫度曲線開始記錄
        ↓
達到 100°C（或按重設）
        ↓
POST /api/auto_temp {enabled:false}
POST /api/fan {fan:3, on:false}
計時器停止
```

## 六、UI 功能清單

| 功能 | 說明 |
|------|------|
| 電磁爐控制 | 手動功率滑桿 + 快捷按鈕（關/20/50/80/MAX） |
| 自動溫控 | ON/OFF 切換，五項參數可調 |

| 風扇控制 | 三路獨立開關（風扇 3 隨自動溫控連動） |
| 電動推桿 | 6 組推桿升降控制 |
| 全畫面切換 | 左下角 ⛶ FAB，適合無鍵盤觸控 |
| 版本管理 | 右下角 🕐 FAB，儲存/還原版本 |
| 系統診斷 | 一鍵診斷硬體連線狀態 |
