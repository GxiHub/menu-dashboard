## Phase 1：基礎設施建置

> 目標：讓 Pi5 能推程式碼到 GitHub

### 1-1 設定 Git 身份

```bash
git config --global user.name "你的名字"
git config --global user.email "你的email@example.com"
git config --global init.defaultBranch main
```

### 1-2 產生 SSH Key

```bash
ssh-keygen -t ed25519 -C "pi53@raspberry"
cat ~/.ssh/id_ed25519.pub
# → 複製公鑰，貼到 GitHub → Settings → SSH Keys
```

### 1-3 安裝 GitHub CLI

```bash
# Raspberry Pi OS (Debian-based)
curl -fsSL https://cli.github.com/packages/githubcli-archive-keyring.gpg \
  | sudo dd of=/usr/share/keyrings/githubcli-archive-keyring.gpg
echo "deb [arch=$(dpkg --print-architecture) \
  signed-by=/usr/share/keyrings/githubcli-archive-keyring.gpg] \
  https://cli.github.com/packages stable main" \
  | sudo tee /etc/apt/sources.list.d/github-cli.list > /dev/null
sudo apt update && sudo apt install gh -y
gh auth login   # 選 SSH → 選已有的 key
```

### 1-4 驗證

```bash
ssh -T git@github.com          # "Hi xxx! You've been authenticated"
gh repo list                   # 列出你的 repos
```

## Phase 4：機密檔集中管理

> 目標：機密檔不散落、不進 Git、有備份

### 4-1 建立集中機密目錄

```bash
mkdir -p ~/secrets
chmod 700 ~/secrets

# 搬移機密檔
mv ~/dashboard/.telegram_env ~/secrets/telegram.env
mv ~/fb_scraper/accounts.json ~/secrets/fb_accounts.json

# 建立 symlink（原位置仍可存取）
ln -s ~/secrets/telegram.env ~/dashboard/.telegram_env
ln -s ~/secrets/fb_accounts.json ~/fb_scraper/accounts.json
```

### 4-2 機密備份（加密）

```bash
# 手動備份（加密 tar）
tar czf - ~/secrets/ | \
  gpg --symmetric --cipher-algo AES256 \
  -o ~/secrets_backup_$(date +%Y%m%d).tar.gz.gpg

# 還原
gpg -d secrets_backup_YYYYMMDD.tar.gz.gpg | tar xzf -
```

### 4-3 機密清單維護

在 `~/secrets/README.md` 記錄：

```
| 檔案 | 用途 | symlink 位置 | 更新頻率 |
|------|------|-------------|---------|
| telegram.env | Watchdog 通知 | ~/dashboard/.telegram_env | 少 |
| fb_accounts.json | FB 爬蟲登入 | ~/fb_scraper/accounts.json | 偶爾 |
```

## Phase 5：自動備份機制

### 5-1 每日本地備份（資料檔）

```bash
#!/bin/bash
# ~/scripts/daily_backup.sh
BACKUP_DIR="/media/usb_backup"  # 外接 USB 或 NAS
DATE=$(date +%Y%m%d)

# 備份資料目錄（不是程式碼，是資料）
tar czf "$BACKUP_DIR/data_$DATE.tar.gz" \
  ~/accounting/data/ \
  ~/fb_scraper/data/ \
  ~/dashboard/logs/ \
  ~/secrets/

# 保留最近 30 天
find "$BACKUP_DIR" -name "data_*.tar.gz" -mtime +30 -delete

echo "[$DATE] Backup done" >> ~/dashboard/logs/backup.log
```

### 5-2 每週 Git 自動推送

```bash
#!/bin/bash
# ~/scripts/weekly_git_push.sh
REPOS=(
  ~/dashboard
  ~/raspi-system
  ~/fb_scraper
  ~/accounting
  ~/projects/robot_arm
  ~/dotfiles_repo
)

for repo in "${REPOS[@]}"; do
  cd "$repo" || continue
  if [ -n "$(git status --porcelain)" ]; then
    git add -A
    git commit -m "auto: weekly snapshot $(date +%Y-%m-%d)"
    git push origin main
    echo "[$(date)] Pushed $repo" >> ~/dashboard/logs/git_push.log
  fi
done
```

### 5-3 Cron 排程

```cron
# 每天凌晨 3 點本地備份
0 3 * * * bash ~/scripts/daily_backup.sh >> ~/dashboard/logs/backup.log 2>&1

# 每週日凌晨 4 點 Git push
0 4 * * 0 bash ~/scripts/weekly_git_push.sh >> ~/dashboard/logs/git_push.log 2>&1
```

### 5-4 備份失敗 Telegram 通知

在備份腳本末尾加入失敗偵測：

```bash
if [ $? -ne 0 ]; then
  source ~/secrets/telegram.env
  MSG="⚠️ 備份失敗: $(hostname) $(date)"
  wget -qO- "https://api.telegram.org/bot${TG_TOKEN}/sendMessage?\
chat_id=${TG_CHAT_ID}&text=$(python3 -c \
"import urllib.parse; print(urllib.parse.quote('$MSG'))")"
fi
```

## Phase 6：災難復原文件

> `RECOVERY.md` — 全新 Pi5 從零重建步驟

### 6-1 系統基礎

```
1. 燒錄 Raspberry Pi OS → SD 卡
2. 設定 hostname: pi53, user: pi53
3. 安裝基礎套件：
   sudo apt update && sudo apt install -y \
     git docker.io docker-compose mosh tmux python3-pip
4. 安裝 Tailscale：
   curl -fsSL https://tailscale.com/install.sh | sh
   sudo tailscale up
5. 安裝 Node.js + Claude Code：
   curl -fsSL https://deb.nodesource.com/setup_22.x | sudo bash -
   sudo apt install -y nodejs
   npm install -g @anthropic-ai/claude-code
```

### 6-2 還原程式碼

```
6. 設定 SSH key → 加到 GitHub
7. Clone 所有 repos：
   gh repo clone your-user/menu-dashboard ~/dashboard
   gh repo clone your-user/raspi-system ~/raspi-system
   gh repo clone your-user/fb-scraper ~/fb_scraper
   gh repo clone your-user/accounting ~/accounting
   gh repo clone your-user/robot-arm ~/projects/robot_arm
   gh repo clone your-user/pi53-dotfiles ~/dotfiles
8. 還原 dotfiles：
   cp ~/dotfiles/.bashrc ~/.bashrc
   cp ~/dotfiles/.tmux.conf ~/.tmux.conf
   cp ~/dotfiles/work.sh ~/work.sh
```

### 6-3 還原機密 & 服務

```
9.  解密還原 secrets：
    gpg -d secrets_backup.tar.gz.gpg | tar xzf -
    建立 symlinks（參照 Phase 4）
10. 啟動 Docker 服務：
    cd ~/raspi-system && docker compose up -d
11. 還原 crontab：
    crontab ~/dotfiles/crontab.txt
12. 驗證：
    ~/health_check.sh
    ~/dashboard/watchdog.sh
```
