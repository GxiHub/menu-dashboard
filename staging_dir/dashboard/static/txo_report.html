系統根據 IV Rank、DTE、組合 Delta、近期事件，輸出建議或自動下單
▶ 每日 15:05 觸發
  ├─ 計算 TXO ATM IV & IV Rank（過去252日）
  ├─ 計算組合總 Delta 偏移量
  ├─ 確認距到期日 DTE
  ├─ 掃描近期事件（Fed / 台股特殊結算）
  └─ 進入決策：
      IF IV Rank > 50% AND DTE >= 21 AND 無重大事件
          → 建議 Short Strangle（最佳履約價自動計算）
      IF IV Rank 30–50% AND DTE >= 21
          → 建議 Iron Condor（保守部位）
      IF IV Rank < 30% OR DTE < 7
          → 不進場，等待更好時機
      持倉監控：50% 獲利停利 / 200% 虧損停損 → 自動平倉

風控規則（8條硬規則）

進場規則

01IV Rank 門檻：Strangle 需 > 50%；Iron Condor 需 30–50%；否則不進場
02DTE 控制：距到期至少 21 天才開新倉，避免 Gamma 風險暴增
03同時部位上限：最多 2 個 Strangle 或 3 個 Condor
04事件迴避：Fed 決策日、台股特殊結算前 3 天不開新倉

停損停利規則

05停利：單筆獲利達收入權利金 50% → 自動平倉，不貪
06停損：單筆虧損達收入權利金 200% → 強制平倉
07Delta 中性：組合 Delta 超過 ±50 → 買期貨中和方向性風險
08黑天鵝保護：台指單日跌超 3% → 系統暫停，等人工確認再進場
